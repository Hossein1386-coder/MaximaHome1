# راهنمای PWA (Progressive Web App) سیستم پذیرش خودرو

این سیستم اکنون به عنوان یک Progressive Web App (PWA) قابل نصب و استفاده است.

## ویژگی‌های PWA

✅ **قابلیت نصب**: می‌توانید این اپلیکیشن را روی دستگاه‌های موبایل و دسکتاپ نصب کنید

✅ **کار آفلاین**: با استفاده از Service Worker، اپلیکیشن می‌تواند در حالت آفلاین نیز کار کند

✅ **کش هوشمند**: فایل‌های استاتیک به صورت خودکار کش می‌شوند تا بارگذاری سریع‌تر شود

✅ **تجربه کاربری بهتر**: اپلیکیشن مانند یک اپلیکیشن بومی عمل می‌کند

## نحوه نصب

### روی موبایل (Android)

1. مرورگر Chrome یا Edge را باز کنید
2. به آدرس `/paziresh/` بروید
3. در منوی مرورگر گزینه "Add to Home Screen" یا "نصب اپلیکیشن" را انتخاب کنید
4. روی "Install" یا "نصب" کلیک کنید

### روی موبایل (iOS - Safari)

1. Safari را باز کنید
2. به آدرس `/paziresh/` بروید
3. دکمه Share (اشتراک‌گذاری) را فشار دهید
4. گزینه "Add to Home Screen" را انتخاب کنید
5. نام اپلیکیشن را تایید و "Add" را بزنید

### روی دسکتاپ (Chrome/Edge)

1. مرورگر Chrome یا Edge را باز کنید
2. به آدرس `/paziresh/` بروید
3. روی آیکون نصب در نوار آدرس کلیک کنید
4. یا از منوی Settings > Apps > Install app استفاده کنید

## فایل‌های PWA

- `manifest.json`: فایل مانیفست که اطلاعات اپلیکیشن را مشخص می‌کند
- `sw.js`: Service Worker که مسئول کش و کار آفلاین است
- لینک‌های manifest و متا تگ‌های PWA در `index.html` اضافه شده‌اند
- کد ثبت Service Worker در `script.js` اضافه شده است

## به‌روزرسانی

Service Worker به صورت خودکار به‌روزرسانی می‌شود. هنگام به‌روزرسانی، یک پیام به کاربر نمایش داده می‌شود که می‌تواند صفحه را رفرش کند.

## تست

برای تست PWA:

1. مرورگر را باز کنید
2. Developer Tools را باز کنید (F12)
3. به تب Application بروید
4. در بخش Service Workers می‌توانید وضعیت Service Worker را ببینید
5. در بخش Manifest می‌توانید اطلاعات مانیفست را ببینید

## نکات مهم

- Service Worker فقط در محیط HTTPS یا localhost کار می‌کند
- برای استفاده از PWA، باید از یک سرور وب استفاده کنید (نه file://)
- در صورت تغییر Service Worker، باید نسخه CACHE_NAME را تغییر دهید

## عیب‌یابی

اگر PWA کار نمی‌کند:

1. بررسی کنید که Service Worker ثبت شده باشد (در Developer Tools)
2. مطمئن شوید که از HTTPS استفاده می‌کنید
3. Cache را پاک کنید و دوباره امتحان کنید
4. بررسی کنید که مسیرهای فایل‌ها درست باشند
